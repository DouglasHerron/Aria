### 3. Vercel deployment

```bash
# Install Vercel CLI
npm install -g vercel

# Deploy
vercel --prod
```

When deploying to Vercel:
1. Set all environment variables from `.env.example` in the Vercel dashboard (Settings ‚Üí Environment Variables)
2. The Obsidian and Apple Calendar integrations will only work locally ‚Äî make sure `DEMO_MODE=true` is set on Vercel for the live demo link
3. Add the Vercel domain to Google OAuth's authorized redirect URIs: `https://your-app.vercel.app/api/auth/callback/google`
